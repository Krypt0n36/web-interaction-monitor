Web Interaction Monitor - Privacy Policy

Last Updated: March 2024

Introduction

Web Interaction Monitor is a Chrome extension designed to track and analyze user interactions on webpages. This privacy policy explains how we collect, use, and protect your data.

Data Collection

Our extension collects the following types of data:

  • Mouse movements (coordinates and timestamps)
  • Scroll events (direction and timestamps)
  • Click events (element information and timestamps)
  • Keypress events (timestamps)

All data is collected locally on your device and is not transmitted to any external servers.

Data Storage

All collected data is stored locally on your device using Chrome's storage API. This includes:

  • Current session interactions
  • Total interaction statistics
  • Download history
  • User preferences (interaction limits)

Data is automatically downloaded as JSON files when the interaction limit is reached.

Data Usage

The collected data is used solely for:

  • Displaying real-time interaction statistics
  • Generating downloadable interaction logs
  • Maintaining total interaction counts
  • Providing session history

Data Protection

We implement the following measures to protect your data:

  • All data is stored locally on your device
  • No data is transmitted to external servers
  • Data is only accessible through the extension's interface
  • Downloaded files are stored in your local downloads folder

Permissions

The extension requires the following permissions:

  • Storage: To save interaction data and preferences locally
  • ActiveTab: To monitor interactions on the current webpage
  • Downloads: To save interaction data as JSON files
  • Host Permissions: To track interactions across different websites

User Control

You have full control over your data:

  • You can view all collected data through the extension's interface
  • You can download your interaction data at any time
  • You can uninstall the extension to remove all stored data
  • You can customize the interaction limit

Third-Party Access

We do not share your data with any third parties. All data remains on your device and is only accessible to you through the extension's interface.

Updates to Privacy Policy

We may update this privacy policy from time to time. Any changes will be reflected in the "Last Updated" date at the top of this document.

Contact

If you have any questions about this privacy policy or the extension's data practices, please contact us through the Chrome Web Store.

Compliance

This extension complies with:

  • Chrome Web Store Developer Program Policies
  • General Data Protection Regulation (GDPR) requirements
  • California Consumer Privacy Act (CCPA) requirements
